Introduction

Enceladus is a http server written in C++. It is compiled as a shared library and can be included in other projects. The goal of this project is to provide a light weight embedded web server that can be easliy included into a linux application.

Setup

There are minimal dependencies. Currently, this is built for linux only. It should not be hard to port to windows. All that's required is g++ and make

$ sudo apt install g++ make

Then clone the repo using this command.

$ git clone https://github.com/shawndfl/Enceladus_Http_Server.git
